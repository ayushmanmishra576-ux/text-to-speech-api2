# 🎙️ Free Text-to-Speech API

A completely free, self-hosted Text-to-Speech API using Microsoft Edge's voices. No API keys, no costs, no limits!

## ✨ Features

- 🆓 **Completely free** - No API keys required
- 🌍 **100+ voices** - Multiple languages and accents
- 🚀 **Fast** - Built with FastAPI for high performance
- 🌐 **Web interface** - Test it directly in your browser
- 📝 **Simple API** - Easy to integrate with any app

## 🚀 Quick Start

### Local Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/my-tts-api
cd my-tts-api

# Install dependencies
pip install -r requirements.txt

# Run the server
python main.py
